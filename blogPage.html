1. What is the difference between forEach map and filter find?


The main difference between forEach and filter is that forEach just a loop over the array and execute the callback but filter executes the callback and check its return value. If the value is true element remains in the resulting array but if the return value is false the element will be removed for the resulting array.

2. Difference between local storage session storage and cookies?


Local storage is useful for storing data that the user will need to access later, such as offline data. Session storage is a great way to improve the performance of your web applications. Cookies are a good choice for storing data that should not be persisted for a long time, such as session.

2. ChooseQuiz function:


The chooseQuiz function is a JavaScript function that is used to update the user's selected answer for a quiz question. First, the function checks if an answer object with the same id as the current quiz question already exists in the answers array using the find method. If no answer object with the same id exists, the function creates a new answer object using the spread syntax to copy all properties of the current quiz question object from the quizData array into a new object, and then adds the givenAns property to the new object with the value of the givenAns parameter. The new answer object is then added to the answers array using the push method.
